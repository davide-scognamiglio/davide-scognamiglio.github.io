ABOUT

My mission
as a Bioinformatician.

Big Data has exploded in the last decade of biomedical research. Sequencing a whole genome is now an accessible task, even at a single-cell level. This high-throughput approach to biomedical research opens for incredible possibilities but also challenges: data analysis among them. By combining biostatistics and programming skills, while keeping a solid biological background, my mission is to apply computational methods towards a healthier society

COVID-19 data visualizer

COVID-19 data visualizer

It started as a procedural CLI application which could plot only few national variables, but it grew bigger as days passed by.
I'm glad I discovered Shiny in the process, since it is a powerful R package
that lets you generate HTML code and interactive standalone web apps.
Thanks to Shiny my homemade visualizer now features an intuitive GUI where you can select:
- Dataset: choose between national or regional dataset
- Regions: if you select regional dataset, a list of regions to compare will appear
- Time range: select by wave or by prompting a custom time range
- Plot to display: pick a line for date-variable plots, or correlation matrix to compare two variables.
- Plot variables: select the variable you want to visualize on the axes or as colour label
